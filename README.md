I build open-source hardware powered by Bitcoin Lightning to make payments tangible, playful, and real. Bilingual (FR/EN) step-by-step tutorials so any maker or educator can reproduce them independently.

Deployed at 8 international Bitcoin conferences across 5 countries. ~1,474 Lightning transactions processed with ~635 participants. Case studies available at [ezmo.dev](https://ezmo.dev).

Contributed to [BTCPay Server](https://github.com/btcpayserver/btcpayserver) (BitcoinSwitch plugin testing), [LNbits](https://github.com/lnbits/lnbits) (production feedback), and [Plan ₿ Academy](https://github.com/PlanB-Network/bitcoin-educational-content) (first bilingual Lightning hardware tutorial on the platform).
